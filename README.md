# GUI для оператора биореактора

Современное приложение для управления биореактором с поддержкой основных функций культивирования микроорганизмов.

## Структура проекта

```
/workspace
├── backend/
│   └── app.py              # Flask REST API сервер
├── frontend/
│   └── gui.py              # GUI приложение на CustomTkinter
└── README.md
```

## Требования

- Python 3.8+
- Flask
- requests
- customtkinter

## Установка зависимостей

```bash
pip install flask requests customtkinter
```

## Запуск

### 1. Запуск бэкенда (в отдельном терминале)

```bash
cd /workspace/backend
python app.py
```

Сервер запустится на `http://localhost:5000`

### 2. Запуск GUI (в другом терминале)

```bash
cd /workspace/frontend
python gui.py
```

## Функциональность

### Мониторинг параметров в реальном времени:
- 🌡️ Температура (°C)
- 🧪 pH уровень
- 💨 Растворенный кислород (%)
- ⚙️ Скорость мешалки (об/мин)
- 📈 Оптическая плотность (OD600)
- 🧫 Объем культуры (мл)
- 💉 Насосы A и B (мл/мин)

### Управление процессом:
- ▶️ **Запуск** - начало культивирования
- ⏸️ **Пауза** - временная остановка процесса
- ⏹️ **Остановка** - завершение культивирования
- 🔄 **Сброс** - возврат к начальным параметрам

### Настройка параметров:
- Установка температуры (20-50°C)
- Установка целевого pH (5.0-9.0)
- Регулировка скорости мешалки (0-1000 об/мин)
- Управление насосом A (кислота/щелочь)
- Управление насосом B (питательная среда)

## REST API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/status` | Получить текущее состояние |
| GET | `/api/history/<param>` | Получить историю параметра |
| POST | `/api/control/start` | Запустить культивирование |
| POST | `/api/control/stop` | Остановить культивирование |
| POST | `/api/control/pause` | Пауза/возобновление |
| POST | `/api/settings/temperature` | Установить температуру |
| POST | `/api/settings/ph` | Установить pH |
| POST | `/api/settings/stirrer` | Установить скорость мешалки |
| POST | `/api/settings/oxygen` | Установить кислород |
| POST | `/api/pumps/a` | Управление насосом A |
| POST | `/api/pumps/b` | Управление насосом B |
| POST | `/api/reset` | Сброс параметров |

## Примечания

- Бэкенд включает симулятор биореактора для демонстрации
- В реальной системе данные поступают от физического оборудования
- GUI автоматически обновляет данные каждые 2 секунды
- При закрытии GUI поток обновления корректно останавливается
